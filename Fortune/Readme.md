## Fortune

使用方式：`./Fortune.py <QQ号>`

抽取运势，每天每用户第一次调用时进行计算，并将结果存入数据库，

之后（同用户）再次调用时，不再进行计算，改为直接从数据库调取当天第一次抽取结果。

数据库为SQLite数据库，位于`./Fortune_Data/Fortune_Data.db`

初次运行时会通过爬虫获取节气信息，当到某个节气时，第一次抽取会返回节气提示.

pip列表：
`pip install requests_html`
